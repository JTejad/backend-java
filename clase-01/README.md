# Clase 01 - Introducción y entorno

## Temas vistos

- presentación del curso y dinámica de trabajo
- instalación de JDK
- instalación de IntelliJ IDEA o Eclipse
- configuración inicial de Git
- creación de un proyecto Java
- primer programa: `HolaMundo`

## Idea principal

En esta primera clase el objetivo fue dejar el entorno preparado para poder programar en Java sin problemas. También vimos la estructura básica de una clase con el método `main`.

## Herramientas mencionadas

- JDK
- IntelliJ IDEA o Eclipse
- Git

## Comandos de Git usados

```bash
git config --global user.name "Tu Nombre"
git config --global user.email "tucorreo@example.com"
```
