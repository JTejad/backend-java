# Clase 08 - Excepciones y modulos

## Temas vistos

- excepciones checked y unchecked
- uso de `try`, `catch` y `finally`
- excepciones personalizadas
- organizacion del codigo con paquetes
- `module-info.java`

## Resumen

En esta clase vimos como manejar errores sin que el programa falle de forma desordenada. Tambien repasamos como separar el codigo en paquetes y como se declara un modulo en Java.

## Ideas importantes

- una excepcion interrumpe el flujo normal del programa
- `try` envuelve codigo que puede fallar
- `catch` permite manejar el error
- `finally` se ejecuta siempre
- las excepciones personalizadas sirven para representar errores del dominio
