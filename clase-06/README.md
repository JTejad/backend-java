# Clase 06 - Encapsulamiento y colaboracion entre clases

## Temas vistos

- encapsulamiento
- visibilidad `public`, `private` y `protected`
- getters y setters
- validaciones en atributos
- objetos dentro de otros objetos
- variables estaticas

## Resumen

En esta clase vimos como proteger mejor el estado interno de los objetos. Tambien empezamos a relacionar clases entre si, por ejemplo un carrito que contiene productos.

## Ideas importantes

- los atributos suelen declararse como `private`
- los getters permiten leer datos
- los setters permiten validar antes de modificar
- las clases pueden colaborar entre si
- una variable `static` pertenece a la clase y no a un objeto puntual
