# Clase 05 - Programacion orientada a objetos

## Temas vistos

- diferencia entre programacion estructurada y POO
- clases, objetos y atributos
- creacion de objetos en Java
- metodos de instancia
- metodos estaticos
- constructores

## Resumen

En esta clase empezamos a modelar el problema con objetos en lugar de trabajar solo con variables sueltas. La idea principal fue representar entidades como productos y clientes con sus propios datos y comportamientos.

## Ideas importantes

- una clase funciona como plantilla
- un objeto es una instancia concreta de una clase
- los atributos guardan el estado del objeto
- los metodos de instancia trabajan sobre un objeto especifico
- los metodos estaticos sirven para logica general compartida
