# Backend Java - Apuntes y ejercicios

Este repositorio lo armé para ir guardando lo que voy viendo clase a clase en el curso de Backend Java.

La idea es usarlo como:

- resumen de teoría
- práctica de ejercicios
- seguimiento de mi progreso

## Estructura

- `clase-01`: introducción al curso, herramientas y primer programa en Java
- `clase-02`: variables, operadores, entrada/salida de datos y estructuras de control

## Nota

Los ejemplos están organizados por clase y tema para que sea más fácil repasar.
