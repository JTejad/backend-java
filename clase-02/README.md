# Clase 02 - Fundamentos de programación en Java

## Temas vistos

- tipos de datos primitivos
- variables
- operadores aritméticos, relacionales y lógicos
- entrada y salida de datos con `Scanner`
- condicionales `if`, `else if`, `else`
- bucles `for` y `while`

## Resumen

En esta clase vimos la base para empezar a resolver problemas con Java. La idea fue trabajar con ejemplos simples relacionados con stock, pedidos y datos de clientes.

## Tipos usados en clase

- `int`: para cantidades o stock
- `double`: para precios
- `boolean`: para validaciones
- `String`: para nombres o mensajes

## Qué me llevo de esta clase

- una variable guarda información para usarla después
- los operadores sirven para calcular y comparar
- `Scanner` permite leer datos desde consola
- los `if` ayudan a tomar decisiones
- los bucles evitan repetir código manualmente
