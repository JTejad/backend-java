# Clase 07 - Herencia y polimorfismo

## Temas vistos

- herencia en Java
- superclases y subclases
- uso de `super` y `this`
- sobrescritura de metodos
- polimorfismo
- clases abstractas e interfaces

## Resumen

En esta clase vimos como evitar duplicacion de codigo creando jerarquias de clases. Tambien trabajamos con polimorfismo para tratar distintos tipos de productos de una forma comun, pero dejando que cada uno tenga su propio comportamiento.

## Ideas importantes

- una subclase hereda atributos y metodos de una superclase
- `super(...)` permite reutilizar el constructor de la clase padre
- `@Override` sirve para redefinir comportamiento en la subclase
- con polimorfismo se puede usar una referencia del tipo padre para objetos hijos
- una interfaz define un contrato que las clases deben cumplir
