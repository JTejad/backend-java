# Clase 04 - Funciones y modularizacion

## Temas vistos

- metodos en Java
- declaracion y llamada de funciones
- parametros y argumentos
- paso por valor
- sobrecarga de metodos
- organizacion del codigo en partes mas chicas

## Resumen

En esta clase vimos por que conviene dividir el programa en metodos. Eso ayuda a reutilizar logica, entender mejor el codigo y hacer cambios sin romper todo el `main`.

## Ideas importantes

- un metodo puede devolver un valor o no devolver nada
- los parametros son los datos que el metodo espera
- los argumentos son los valores que se le pasan al llamarlo
- en Java los valores se pasan por valor
- la sobrecarga permite usar el mismo nombre con distintas firmas
