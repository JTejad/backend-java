# Clase 03 - Cadenas y listas

## Temas vistos

- clase `String`
- métodos comunes de cadenas
- limpieza y formateo de texto
- arrays
- listas dinámicas con `ArrayList`
- recorrido y actualización de colecciones

## Resumen

En esta clase vimos cómo trabajar mejor con texto y con grupos de datos. Primero usamos cadenas para limpiar nombres de productos y después pasamos a arrays y `ArrayList` para manejar catálogos de forma más ordenada.

## Ideas importantes

- las cadenas en Java son inmutables
- los métodos como `trim()`, `toLowerCase()` y `substring()` devuelven una nueva cadena
- un array tiene tamaño fijo
- `ArrayList` permite agregar y eliminar elementos más fácilmente
